
**CS 351L - AI Lab**

Welcome to the AI Lab Course (CS 351L)! In this repository, you will explore various concepts in Artificial Intelligence (AI) through hands-on exercises and projects using Python. The course focuses on practical implementations of AI techniques, algorithms, and tools commonly used in the field of AI and cybersecurity.

**📋 Course Outline**

Throughout the course, we will cover the following topics:

2- Introduction to Python: Variables, Data Types, and Control Structures

3- AI Development Environment: Setting up Google Colab, Introduction to NumPy, Pandas, Matplotlib

4- Supervised Learning: Linear Regression, Classification

5- Unsupervised Learning: K-Means, Hierarchical Clustering

6- Neural Networks: Introduction and Implementation

7- Evaluation Metrics: Precision, Recall, F1-Score

8- Hands-on Projects: AI techniques applied to real-world cybersecurity problems

9- Tool Use: WEKA for data mining and machine learning tasks
